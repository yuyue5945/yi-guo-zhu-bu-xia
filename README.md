# 一锅煮不下 🍲

火锅主题垂直自由落体消除小游戏。

## 玩法

1. 锅上方随机出现一个食材，点击让它自由落体
2. 食材从锅中央垂直下落，遇到阻挡自动停止
3. 相同食材在横竖方向≥3个时自动消除
4. 达到目标分数过关
5. 堆满锅顶则游戏结束

## 技术栈

- HTML5 Canvas
- Web Audio API（原创火锅主题BGM）
- 纯前端，无依赖

## 开发

```bash
# 本地预览
python3 -m http.server 8080
# 打开 http://localhost:8080
```

## 部署
